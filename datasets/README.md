#  Data Folder

This folder contains all datasets used for analyzing trading behavior and market sentiment.

##  Datasets Included

* `historical_data.csv` → Trading activity data (volume, leverage, PnL, etc.)
* `fear_greed_index.csv` → Market sentiment data (Fear vs Greed index)

---

##  Purpose

These datasets are used to:

* Analyze trader behavior under different market sentiments
* Study the relationship between sentiment and profitability
* Generate insights for potential trading strategies

---

##  Notes

* Raw datasets are kept unchanged for reproducibility
* Processed datasets are stored separately (if applicable)
* Avoid uploading extremely large datasets

---

##  Data Processing

* Merging datasets based on time
* Handling missing values
* Normalizing/cleaning inconsistent entries
